Algolia

Docsearch​

在Docusaurus專案底下安裝docsearch

npm install @docsearch/react

Algolia​

註冊​

到Algolia官方網站註冊並登入

建立index​

之後會用到index名稱，可以先複製下來

API Key​

  1. 點選API Keys
  2. 點選All API Keys
  3. 點選New API Key
  4. 在最下方的ACL額外新增addObject editSettings deleteIndex 三個參數
  5. 點選Create

Docusaurus Config​

在docusaurus.config.js的themeConfig中新增algolia物件

  • apiKey: 剛剛新增的API Key
  • indexName: 剛剛建立的index名稱
  • apiId: 在API Key的前一頁可以找到Application ID
...
themeConfig: ({
    algolia: {
        apiKey: "a54fa7cc6c9ecfbcd49ef66fd9970449",
        indexName: "dev_docusaurus",
        appId: "UF84S2O5HS",
    },
    ...
});

設定完後會在網頁右上角看到搜尋欄

.env​

在Docusaurus專案的根目錄新增.env:

//.env
APPLICATION_ID=userApplicationId
API_KEY=userAPIKey

Docker Desktop/ jq​

  1. 安裝Docker Desktop
  2. 安裝chocolately，以系統管理員開啟CMD，輸入以下指令:
@"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
  1. 用chocolatey安裝jq:
chocolatey install jq

爬取資料新增至Algolia​

用Git Bash開啟Docusaurus專案，並輸入:

docker run -it --env-file=.env -e "CONFIG=$(cat ./config.json | jq -r tostring)" algolia/docsearch-scraper

Windows作業系統如果有跳出winpty提示，則改為輸入:

winpty docker run -it --env-file=.env -e "CONFIG=$(cat ./config.json | jq -r tostring)" algolia/docsearch-scraper

執行成功時，docsearch-scraper會爬取網站內容並新增至Algolia，回到Algolia後台的Search頁面可以看到網站被新增成功，此時就可以使用Docusaurus的搜尋欄了!
